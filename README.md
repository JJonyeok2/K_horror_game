# K_horror_game

K-호러 테마의 회수 생존 게임 프로토타입입니다. `Lethal Company`류의 할당량/회수/탈출 루프를 한국적 장소, 문화재급 회수품, 금기, 귀신/요괴/민속 괴이로 재해석합니다.

## MVP

- 엔진: Godot 4.x
- 플레이 방식: 1인칭 싱글플레이
- 첫 맵: 종가 고택
- 시작/복귀 허브: 대문 밖에 세워진 낡은 회수업체 봉고풍 트럭
- 핵심 루프: 봉고 옆 시작 -> 고택 대문 진입 -> 회수품 탐색 -> 봉고 적재함에 물건 적재 -> 재진입 또는 철수
- 위협 루프: 회수품 획득 -> 원한 수치 상승 -> 사운드/위협 강화 -> 대문 밖 복귀 압박
- 정산 루프: 소지품은 2칸 제한이며, 봉고 적재함에 넣은 물건만 회수 금액으로 확정

## 탐험/복귀 구조

플레이어는 고택 안에서 시작하지 않고, 대문 밖 회수업체 트럭 옆에서 시작합니다. 트럭은 오래된 흰색 소형 화물차, 즉 옛날 봉고 느낌의 무표식 회수 차량입니다. 플레이어는 대문을 지나 종가 고택 안으로 들어가고, 회수품을 최대 2개까지 들고 다시 대문 밖으로 돌아옵니다.

대문 밖으로 나왔다고 바로 임무가 끝나지는 않습니다. 물건은 트럭 적재함에 넣어야 안전하게 확보되며, 확보된 물건만 할당량 정산에 반영됩니다. 이후 플레이어는 다시 들어가 더 가져올지, 운전석/무전기에서 철수할지 선택합니다.

MVP 구현 목표:

- 대문 밖 시작 위치
- 고택 대문을 통한 탐험 구역 진입
- 봉고풍 회수 차량과 적재함
- 소지품 2칸에서 적재함으로 물건 옮기기
- 차량에서 철수 상호작용
- 철수 시 할당량 달성/미달 결과 표시

## 실행

```bash
godot --path .
```

## 조작

- 이동: `WASD`
- 점프: `Space`
- 회수/상호작용: `E`
- 소지품: 2칸 제한
